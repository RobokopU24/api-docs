ROBOKOP Knowledge Graph (KG) is an open-source biomedical knowledge graph that supports the ROBOKOP application and currently contains ~ 10M nodes and ~ 250M edges drawn from ~30 integrated and harmonized biological data sources and bio-ontologies.

**Version:** b0ed1e26f7f3726e

**URL:** [https://robokop.renci.org/](https://robokop.renci.org/)

**Neo4J:** [https://stars.renci.org/var/plater/bl-3.2.1/RobokopKG/b0ed1e26f7f3726e/graph_b0ed1e26f7f3726e.db.dump](https://stars.renci.org/var/plater/bl-3.2.1/RobokopKG/b0ed1e26f7f3726e/graph_b0ed1e26f7f3726e.db.dump)

**Nodes:** 8,660,052

**Edges:** 132,035,404
